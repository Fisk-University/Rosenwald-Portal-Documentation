# AWS S3 Bucket Setup Tutorial 

## Step 1: Log in to AWS
- Visit: [aws.com](https://console.aws.amazon.com)
- Use an IAM username with appropriate permissions to sign in

## Step 2: Create a New Bucket
- Go to the search bar and look up s3
- Select s3 and click **Create bucket**
- Provide a unique name for the bucket(e.g., `omeka-archive-bucket`)
- Choose a region (United States N. Virginia) 

## Step 3: Set General Configuration
- Set Bucket type to General Purpose
- Set Object Ownership to ACLs enabled
- Deselect "block all public access" and click to acknowledge the warning 
- Bucket versioning should be set to Disable
- Default encryption and Bucket key should be let as they are
- Click on "Create bucket"
