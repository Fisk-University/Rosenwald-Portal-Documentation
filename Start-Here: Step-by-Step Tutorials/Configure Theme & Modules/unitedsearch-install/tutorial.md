# UnitedSearch Module Installation and Activation Tutorial

 This tutorial will walk you through the steps to install and activate **UnitedSearch** module for Omeka S.

## Requirements
> ** Note:** UnitedSearch requires **Omeka S version 4.0.0 or higher** to work properly.

---

## 1. Download UnitedSearch

- Download the [latest release](https://github.com/Fisk-University/UnitedSearch/releases) of UnitedSearch.

---
## 2. Prepare the Module
- Extract (Unzip) the downloaded '.zip' file.
- Rename the directory to "UnitedSearch"

---
## 3. Upload to Omeka S
- Upload the renamed 'UnitedSearch' directory into your **Omeka S modules directory**, (usually at /modules)

---
## 4. Upload to Omeka S

You can upload the module using either **FTP** or **SSH**:

###  FTP Method

1. Open your FTP client, log on to the server that houses the Omeka S install.
2. Navigate to the `/modules` folder (this should be located in the main folder of the install)
3. Upload the `UnitedSearch` folder.
4. Ensure the directory structure is correct
---
## 5. Activate in Omeka S Admin
- Log in to your **Omeka S admin dashboard**.
- Go to the **Modules** section.
- Find **UnittedSearch** in the list and click **installation**.
- Once installed, clcik **Activate** (if not already activated).

---
UnitedSearch should now be active and ready to use within your Omeka S site.
